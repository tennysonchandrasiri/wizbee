# Twootle English School

Welcome to **Twootle English School**, an interactive and fun English learning experience designed specifically for young learners in Sri Lanka!

<img src="src/assets/icon.png" alt="Twootle Mascot" width="200" />

## About Twootle

Twootle English School is a desktop application that brings English language learning to life through:
- **Interactive Games**: Learning made fun with sorting games, puzzles, and more.
- **Engaging Stories**: Carefully curated stories to improve reading and comprehension.
- **Vocabulary & Grammar**: Comprehensive lessons covering essential English foundations.
- **Visual Learning**: Beautiful picture descriptions and colorful interfaces.

**"The eye to see the world through"**

## Features

- **Kid-Friendly Interface**: Designed with a clean blue theme that is inviting and easy to navigate.
- **Offline Capable**: Once installed, your child can learn without an internet connection.
- **Progressive Levels**: Content tailored for different skill levels, from "Twootle Kids" to "Senior Twootle Kids".
- **Safe Environment**: A focused educational tool with no ads or distractions.

## How to Get It

You can download the latest version of Twootle English School for your computer from our [official website](https://tennysonchandrasiri.github.io/twootle/).

- **Windows**: Download the `.exe` installer.
- **macOS**: Download the `.dmg` file.
