## Features

- **Kid-Friendly Interface**: Designed with a warm "honey bee" theme that is inviting and easy to navigate.
- **Offline Capable**: Once installed, your child can learn without an internet connection.
- **Progressive Levels**: Content tailored for different skill levels, from "Wizbee Kids" to "Senior Wizbee Kids".
- **Safe Environment**: A focused educational tool with no ads or distractions.

## How to Get It

You can download the latest version of Wizbee English School for your computer from our [official website](https://tennysonchandrasiri.github.io/wizbee/).

- **Windows**: Download the `.exe` installer.
- **macOS**: Download the `.dmg` file.
