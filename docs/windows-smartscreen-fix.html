How to Configure

  1. Obtain Certificate: Purchase from a trusted Certificate Authority (CA) and complete their validation process (typically 1-3 days).
  2. Export Certificate: For regular certificates, export as a .pfx file with the private key.
  3. Update package.json:
    { "win": { "certificateFile": "path/to/your/certificate.pfx", "certificatePassword": "your_certificate_password", "certificateSubjectName": "Twootle" } }
  4. Build: Run npm run dist:win to create signed installer.

Testing Your Signed App

Verification Steps:

  1. Upload the signed installer to a web server
  2. Download from a different Windows machine (not your development PC)
  3. Verify that no SmartScreen warning appears
  4. Check that the publisher shows as "Twootle" in file properties
